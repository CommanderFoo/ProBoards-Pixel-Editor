Pixel Editor 1.0.0 Beta
=======================


* Undo & Redo
* Choose pixel color
* Fill canvas with selected color
* Color picker
* Show / Hide grid
* Mirror mode
* Erase color
* Save (uses localStorage)
* Save as
* File list
* Export / Import Raw data (not canvas data)
* Basic help
* Drag & Drop onto post
* Double click on image opens options
* Trim image to remove wasted space
